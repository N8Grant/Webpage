Projects


Pacman DDQN


In this project I used the Opan AI gym environment for Ms. Pacman to create a Double Deep Q-Learning Neural Net (aka. DDQN). Initial implementations of the RL algorithm didnt work too well as the Neural Network was over estimating the Q values for states and it plateaued fairly early. So to combat this I increased the buffer size and also implemented another Neural Net to predict the Q values of each state. In order to avoid over-estimation I would take the highest Q value from each and then take the smallest of those two values to be the Q value approximation. Further improvements could include using states for more than just the next timestep. Instead, I would use the unrolling technique to compute errors of Q values from 2-4 steps ahead.



Covid-19 Prediction


In this project I used the John Hopkins Covid-19 data to try to predict the spread of the outbreak. Since county data wasnt available I distributed cases according to population. I then used a spatial SIR model to predict the spreading of the disease. I included the effects of travel, favoring big cities, by giving some random probability that someone gets infected. I plan on accounting for lockdown by tapering off the probability of traveling infectors. I also plan on including hospital data to see which states might be at a particularly high risk due to number of cases w.r.t. to hospital beds.



Cancer Classifier


In this project I used a CNN to predict whether a picture of a skin lesion was cancerous or not using the MNIST-Skin cancer dataset. I was able to get about a 79% accuracy, however this can be further improved. Due to memory constraints I couldn't load all of the pictures for training data, so only about 3/4 of the data was actually used. Another problem is that there was an imbalance in the non-cancerous and cancerous classes. This makes it so that the classifier was trained to slelect the non-cancerous class because it was the most probable.
