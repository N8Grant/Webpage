MNIST Skin Cancer Classifier

This project uses a CNN to classify images of skin based
off wether or not they contain cancer. The model uses two
convolutional layers and then two fully connected layers.
The resulting model ended up gettin an accuracy of 79% on the test set.
In the future it might be worth using an autoencoder as the input so
that the input dimension is a smaller size.
See it On GitHub

Pacman DQN

This project uses a Vanilla DQN to try to beat
the Atari 2600 game Ms Pacman.
The model uses teo convolutional layers and two fully connected layers.
Using a replay buffer and a linearly decreaing epislon,
the model was able to converge to beat the game after over 10,000 games.
See it On GitHub
